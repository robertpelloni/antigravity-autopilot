# Antigravity Autopilot (Unified)

The ultimate AI coding assistant. Merges Auto-Accept, Auto-All, and Yoke Autonomous Mode into one powerful, configurable extension.

## Features
- **Auto-Accept**: Automatically accepts pending inline completions.
- **Auto-All (CDP)**: Robust browser-based automation for multi-tab editing.
- **Autonomous Mode (Yoke)**: Set a goal and let the agent loop until completion.
- **Status Bar Integration**: Visual feedback on active modes.

## Usage
1.  **Toggle Extension**: `Ctrl+Shift+P` -> `Antigravity: Toggle ON/OFF`
2.  **Settings**: `Ctrl+,` -> Search for `antigravity`
